# Predicting Tornado Days from Environmental Parameters

Machine learning project predicting tornado occurrence in Kansas using ERA5 reanalysis data and NOAA storm records.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Open `project.ipynb` and run cells sequentially. Data is automatically downloaded and cached.

## Models

- Logistic Regression
- Random Forest
- Gradient Boosting
- Neural Network

## Data Sources

- **ERA5 Reanalysis**: CAPE, temperature, dewpoint, winds, pressure
- **NOAA Storm Events**: Historical tornado records
